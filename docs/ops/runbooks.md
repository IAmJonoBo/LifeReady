# Operations Runbooks

## Scope

This runbook set covers incident response, rollback, readiness failures,
backup/restore verification, and audit-chain integrity alerts.

## Minimum runbooks required

- Service outage and dependency degradation
- Authentication/key-rotation incident
- Export/storage incident
- Audit-chain mismatch incident
- Rollback and release abort
- Backup restore drill execution
